### Свойства
| Имя                         | Тип                  |
| :-------------------------- | :------------------- |
| **[peerId](#peerid)**       | int                  |
| **[userId](#userid)**       | int                  | 
| **[audio](#audio)**         | Boolean              | 

## Свойства

### peerId
**Тип: int**  
ID диалога, в котором пользователь набирает сообщение 


### userId
**Тип: int**  
ID пользователя, который набирает сообщение


### audio
**Тип: Boolean**  
**true**, если пользователь записывает аудиосообщение, **false**, если текстовое
